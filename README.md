Hello, This is my first time making a custom cursor pack :) 
Just a simple cursor pack !

# To download it :

-Download the latest version from [Releases](https://github.com/xMIMBOx/MIMBO-Cursor-Pack/releases) as a ZIP and extract it.

# To install it :

-Open the folder u just extracted "MIMBO'S PACK"

-Simply "Right Click" The "install.inf"

-in the menu that will pop-up just Click install thats it.


# To actually see/apply the Curser go to :

Controle panel >  Hardware and Sound > Mouse (under Devices and Printers) > Pointers > Scheme > MIMBO'S Custom Pack,

then hit apply. 

thats it.
